## Context

- Current status: !`jj status`
- Add commit description !`jj describe -m "Commit description"`
- Current diff from `main`: !`jj diff -f main`
- Current commits from `main`: !`jj log -r main::`
- Push current changes to upstream: !`jj git push -c 'main+::~description(exact:"")'`
- Get the automatically created bookmark name, should be something like `push-*`: !`jj bookmark list -r 'main+::'`

## Your task

Based on the above changes:
1. Add commit description if empty with appropriate commit message
2. Push the change to upstream and get autogenerated bookmark name
3. Create a PR using the GitHub CLI
